# 🌌 SHADOWX RGB - Background Remover

A simple web app to remove image backgrounds with a cool galaxy theme. ✨

## 🚀 Features

- 🖼️ Remove background from images
- 📤 Upload image or use URL
- ⬇️ Download processed images
```
## ⚡ Quick Setup

1. **Install dependencies**
```bash
pip install -r requirements.txt
```

1. Run the app

```bash
python app.py
```
